C语言指针学习笔记

发表于 2017-03-09

指针(pointer)——保存内存地址的变量
指针是一个值为内存地址的”变量”(或数据对象)
int year;
year = 2016;
year是内存地址的名字。内存里存放的是2016
int ptr_year; 变量类型为int 变量名称为ptr_year
ptr_year = &year; 变量year的内存地址取出来赋值给ptr_year
指针int ptr_year指向int变量 year

指针使用示例


int * ptr_num2;
ptr_num2 = ptr_num;//两个指针指向同一块内存

  • ptr_num2 = 99;//等价写法:num=99;

指针是一个变量，该变量中存储的是另一个对象的内存地址
如果一个变量存储另一个对象的地址，则称该变量指向这个对象
指针变量可以赋值，指针的指向在程序执行中可以改变

  1. 指针变量的命名规则和其他变量的命名规则一样
  2. 指针不能与现有变量同名
  3. 指针可存放C语言中的任何基本数据类型、数组和其他所有高级数据结构的地址
  4. 若指针已声明为指向某种类型数据的地址，则它不能用于存储其他类型数据的地址
  5. 应为指针指定一个地址后，才能在语句中使用指针。//一定不是赋常量，而是用&符号赋地址

num[i] == *(num+i)

&取地址符
*根据地址取值
指针不可以赋常量
指针是一个变量，存储另一个变量（对象）的内存地址
指针的声明由基本类型、星号和变量名组成
为指针赋值，赋值运算符右侧必须是一个地址
若是普通变量想乣在前面加一个取地址运算符&
